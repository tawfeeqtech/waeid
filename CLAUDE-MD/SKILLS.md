# Core Skills System

## 1. salla-component-builder
Builds reusable UI components with:
- Variants
- Responsive behavior
- Token-based styling
- RTL support

---

## 2. salla-settings-engine
Responsible for:
- Converting components into configurable units
- Creating clean settings schema
- Mapping settings to tokens

---

## 3. salla-design-system
Handles:
- CSS variables (tokens)
- Theme presets
- Typography
- Spacing system

---

## 4. salla-rtl-check
Ensures:
- RTL/LTR compatibility
- Logical CSS usage
- Layout correctness

---

## 5. salla-performance
Handles:
- Lazy loading
- DOM optimization
- Media optimization

---

## 6. salla-section-composer
Combines:
- Components into sections
- Sections into pages
- Ensures consistency
