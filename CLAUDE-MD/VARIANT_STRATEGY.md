# Variant Strategy (CRITICAL)

## Problem

Current system limits variants (e.g., 6 hero styles)

This is WRONG.

---

## Solution

Variants should be:

### 1. Pattern-based
Not fixed layouts

Example:
- alignment: center / left / split
- media: image / video / none
- emphasis: text / media / stats

---

### 2. Generated, not counted

Instead of:
6 hero variants ❌

We use:
combinations → 50+ possibilities ✅

---

## Hero System Example

Hero =

- layout_mode
- media_type
- content_alignment
- emphasis
- CTA_style

Each combination = new variant

---

## Goal

Unlimited flexibility without writing new components
