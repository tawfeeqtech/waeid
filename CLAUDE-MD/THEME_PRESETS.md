# Theme Presets System

## Current Issue

Presets are fixed (e.g., 6 styles)

---

## Required Upgrade

Presets should be:

- Expandable
- Data-driven
- Not limited in number

---

## Each Preset Controls:

- Colors
- Shadows
- Border radius
- Typography feel

---

## Important Rule

Presets should NOT block manual overrides

---

## Future Vision

User can:
- Pick preset
- Then customize further

---

## Goal

Make presets:
- Fast start
- Not a limitation
