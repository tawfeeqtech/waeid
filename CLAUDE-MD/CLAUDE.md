# Salla Coach Theme Engine – Master System

## Overview
This project is a **highly flexible Salla Theme Engine** designed for:
- Fitness coaches
- Trainers
- Digital service providers

The goal is NOT to build static themes,
but to build a **configurable UI system** capable of generating many layouts from a single codebase.

---

## Core Philosophy

- System > Pages
- Components > Sections
- Variants > Hardcoded layouts
- Tokens > Raw values
- Configuration > Manual edits

---

## Current System Capabilities

- Smart Hero System (multi-variant, partially configurable)
- Header Variant Matrix (multiple layout combinations)
- Theme Presets (brand identity system using CSS variables)

---

## Current Limitations (MUST FIX)

- Not all components are fully connected to settings
- Theme presets are limited (should be scalable, not fixed)
- Hero customization is limited (colors, layouts, flexibility)
- Variants are manually limited instead of system-generated

---

## Ultimate Goal

Build a **world-class theme system** where:

- Any section can have unlimited variants
- Any component can be fully styled via tokens
- New layouts can be generated without rewriting code
- The merchant feels like they have “everything” available

---

## Golden Rules

1. NEVER hardcode UI decisions
2. ALWAYS use design tokens
3. ALWAYS support RTL + mobile
4. ALWAYS prefer scalable systems over fixed options
5. DO NOT artificially limit variants (no fixed 6, 10, etc.)
