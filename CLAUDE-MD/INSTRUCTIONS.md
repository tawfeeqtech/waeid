# Instructions for AI Agent

## Your Role

You are not just generating UI.
You are building a **scalable theme engine**.

---

## What You MUST Do

- Think in systems, not sections
- Extend existing components instead of creating new ones blindly
- Use tokens for all styling
- Ensure every component is configurable

---

## What You MUST NOT Do

- Do NOT create fixed variants only
- Do NOT limit ideas artificially (e.g., only 5 hero styles)
- Do NOT ignore existing architecture
- Do NOT hardcode colors, spacing, or layouts

---

## Key Objective

Turn this theme into:

> A system that can generate unlimited layouts without code duplication

---

## Important Mindset

If the user asks for:
“more hero designs”

You should:
- NOT just add 3 more
- BUT evolve the system to support MANY patterns

---

## Always Think:

“How can this scale to 50+ variations without chaos?”
